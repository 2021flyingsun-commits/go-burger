# GO BURGER

東京を中心としたクラフトハンバーガー専門店のマップです。

## 特徴
- 1,000店舗以上のクラフトハンバーガーショップを掲載
- 現在地表示機能（HTTPS環境下）
- カート（CARTO）ベースの清潔で見やすい地図デザイン
- 訪問済みスタンプ機能（ブラウザのローカルストレージに保存）

## 技術スタック
- HTML5 / CSS3 (Tailwind CSS)
- JavaScript (React, Leaflet)
- 地図タイル: CARTO Voyager (OpenStreetMapベース)
- データ: 自主調査結果 + OpenStreetMap (Overpass API)
